# *Laravel Maps App* 🌍✈️

Projeto Acadêmico – Sistema de gerenciamento e visualização de locais com mapa interativo, desenvolvido como atividade prática de curso.
Diferencial: o sistema conta com uma animação de aviãozinho passando na tela, trazendo dinamismo e personalidade à aplicação. 🛫✨

Este projeto foi desenvolvido em Laravel e integra Leaflet.js (OpenStreetMap) para exibir localizações de forma interativa. Além da funcionalidade técnica, o sistema permite personalização visual, botões destacados, animações criativas e organização dos locais cadastrados.

### Objetivo do Projeto 🎯

Integrar uma aplicação Laravel com um mapa interativo.

Permitir adicionar, visualizar, editar e excluir locais.

Criar uma interface agradável, funcional e personalizada.

Adicionar elementos de criatividade como a animação do aviãozinho.

Aplicar conceitos de desenvolvimento web e banco de dados em um projeto real.

### Funcionalidades 🚀

Mapa Interativo: Visualização de todos os locais cadastrados.

Cadastro de Local: Nome, categoria, endereço, latitude, longitude e descrição.

Lista de Locais: Botões para visualizar, editar e excluir cada local.

Botão “Adicionar Local” destacado em vermelho dentro da lista.

Estilo personalizado: Cores, ícones FontAwesome e interface responsiva.

Animação de Aviãozinho: Um avião cruza a tela em movimento, simulando viagem entre os locais. 🛩️

### Tecnologias Utilizadas 💻

Laravel – PHP Framework

Bootstrap 5 – Framework CSS

Leaflet.js – Mapas interativos

FontAwesome – Ícones

MySQL – Banco de dados

### Como Usar 🗺️

Adicionar Local: Clique no botão vermelho na lista ou no topo (out-line) para criar um novo local.

Preencher informações: Nome, categoria, endereço, latitude e longitude.

Visualizar detalhes: Use o botão “Ver Detalhes” na lista ou no mapa.

Editar ou Excluir: Botões de edição e exclusão em cada item da lista.

Mapa: Marcadores aparecem automaticamente ao adicionar novos locais.

Animação: Observe o aviãozinho passando pela tela representando movimento e viagem.
